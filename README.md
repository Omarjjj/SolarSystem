# Solar System WebGL

Open `index.html` in a browser to view the solar system simulation.
WebGL logic is in `webgl.js`, bootstrap in `main.js`, and shaders live in `shaders/`.

If your browser blocks local files, run a local server from this folder:

- Windows: `start-server.bat`

```bash
python -m http.server
```

Then visit `http://localhost:8000/`.
